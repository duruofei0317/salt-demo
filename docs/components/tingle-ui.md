## 添加组件
* 安装

```
tnpm i @ali/tingle-xxx --save
```
* 在 src/components.sty 里引入对应 stylus

```
@require "./../node_modules/@ali/tingle-xxx/src/Xxx.styl";
```

* 修改 demo, 添加对应组件

* 构建样式

```
gulp b
```

* 查看 demo

```
gulp d
```

## 添加主题

* 在 tingle-style 中添加对应主题变量
* 在 src/theme/添加对应 主题.styl
* 构建样式

```
gulp b
```

* 查看 demo

```
gulp d
```
